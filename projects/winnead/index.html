Winnead - Native Windows Text Editor

Description

A native Windows text editor made with Tauri, React, NPM, and Monaco Editor. It includes built-in features such as a file explorer tree and a terminal with PowerShell.

Features

  1. Simple and Elegant UI.
  2. File Explorer
  3. Integrated terminal with PowerShell

Build from source

git clone https://github.com/hyouteki/Winnead
cd Winnead
npm run tauri dev
